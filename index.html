LIBRARY 01 METHOD
FROM FILES TO STRUCTURE
建设方法

把产品资产
变成内容燃料

产品图、卖点、人群、场景、竞品差异,先标准化。

IMAGE
产品图归档主图、细节图、使用图统一入库
POINT
销售主推卖点功效、参数、证据、禁用词分开
AUDIENCE
目标客户分层谁在买、为什么买、什么场景买
SCENE
成交场景标签门店、直播间、私域、销售跟进
DIFF
竞品差异明确不能说什么，也明确要突出什么
READY
可生成素材包让后面的分镜和提示词有依据

LIBRARY 02

不是抄爆款,是沉淀被市场验证过的成交表达。

COPY STRUCTURE

KEEP BOUNDARY

09 / 22
LIBRARY 02 METHOD
REFERENCE WITH BOUNDARIES
使用方法

参考视频库
怎么用

借开头、借节奏、借转化结构,不借品牌。

01
借开头客户为什么要停下来，看前三秒
02
借动作产品怎么出现，用户怎么使用
03
借节奏哪里加速、哪里停顿、哪里给证明
04
借转化结构痛点、产品、证明、行动怎么串
STOP
不借品牌不复制 IP、口号、人物和独家视觉资产

STORYBOARD METHOD
ONE CLAIM / FOUR SHOTS
成交镜头

一个卖点如何
变成成交镜头

01痛点唤起

先让客户看到自己的问题，不急着介绍产品。

情绪场景
02产品出现

产品以解决方案出现，画面主体明确。

主体卖点
03使用证明

动作展示怎么用，镜头负责证明过程。

动作证据
04行动收口

最后给结果、对比或下一步转化动作。

证明转化
